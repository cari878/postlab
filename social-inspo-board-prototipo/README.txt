# Social Inspo Board

Prototipo stile Pinterest per social media manager.

## Come aprirla
1. Estrai lo ZIP.
2. Apri `index.html` con Chrome, Safari o Edge.
3. Puoi caricare immagini, cercarle, filtrare per formato e salvarle nel browser.

## Funzioni incluse
- Griglia stile Pinterest.
- Upload immagini.
- Ricerca per parole chiave.
- Filtri per Stories, Reel, Caroselli, Post e Adv.
- Board per settore.
- Scheda dettaglio con note.
- Generatore semplice di spunti per adattare una reference a un altro settore.
- Salvataggio locale con localStorage.

## Nota
Questa è una demo offline. Per farla diventare una vera app online servono:
- login utenti;
- database;
- hosting;
- storage immagini;
- sistema di permessi;
- AI collegata via API.
